Último Projeto

Este projeto consiste na criação de uma Landing Page para uma barbearia, utilizando ReactJS para o frontend e SCSS para a estilização. Essas tecnologias garantem uma interface responsiva e de fácil navegação, além de uma organização eficiente e modular do código CSS. Integrado com o Supabase, uma plataforma baseada em PostgreSQL com API RESTful, o projeto permite armazenar e acessar dinamicamente informações sobre os serviços da barbearia. Isso proporciona uma gestão simplificada dos conteúdos exibidos, com atualizações rápidas e seguras, garantindo que os dados estejam sempre atualizados e precisos para uma experiência intuitiva e otimizada para os clientes.
